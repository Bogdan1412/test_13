Задача 4. С использованием замыканий разработать итератор, то есть функцию, которая будет постепенно по отдельным вызовам выдавать по одной букве заранее заданного слова. С использованием него разработать игру «Преводитель». Выводим пользовательское описание этого слова и по одну букву пользователь вводит перевод этого слова. Подсчитать количество упомянутых букв.
